❓ 도움말 & 사용 가이드
🗂️ 다이어그램 모드 (상단 탭)
FC — 플로우차트: 일반 업무/게임 흐름
FSM — 유한 상태 기계: 상태 전환 설계
BT — 비헤이비어 트리: AI 행동 로직
SC — 시퀀스 차트: 객체 간 메시지 흐름
🧩 노드 추가
• 왼쪽 팔레트에서 드래그 → 캔버스에 드롭하거나, 팔레트 항목 클릭 → 화면 중앙에 생성
• 노드 선택 후 오른쪽 인스펙터에서 레이블·크기·색상·속성 편집
• 노드 더블클릭 → 레이블 빠른 수정 (또는 F2)
🔗 노드 연결
• 노드에 마우스를 올리면 상·하·좌·우에 파란 앵커(●)가 표시됨
• 앵커 드래그 → 대상 노드 위에서 손을 떼면 연결 완성
• 대상 노드의 어느 부분에 떨어뜨려도 가장 가까운 앵커로 자동 스냅
• 이미 연결된 선의 끝점 핸들 드래그 → 다른 앵커로 재연결
• 연결 중 Esc 또는 빈 곳 클릭 → 취소
➡️ 선(엣지) 조작
• 상단 🔀 선 스타일 버튼으로 꺾은선 / 직선 / 곡선 전환 (단축키 L)
• 선 선택 후 주황 핸들 드래그 → 경로 세부 조절
• 선 선택 → 인스펙터에서 레이블(조건식)·이름·색상 설정
• 엣지 레이블에 변수명 > 값 형식의 조건식을 입력하면 로직 트레이스에서 자동 평가
📐 조건식 문법 (엣지 레이블)
level >= 5　숫자 비교 (> >= < <=)
state == idle　문자열 동등 비교 (== !=)
isAlive　　　　변수 truthy 체크 (값이 있으면 참)
Yes / No　　　리터럴 참·거짓 (변수명 우선)
※ 변수명은 노드 속성 키 또는 로직 트레이스 변수 패널에 입력
🖱️ 뷰 & 선택
휠 스크롤: 줌 인/아웃
우클릭/중클릭 드래그: 캔버스 이동(Pan)
바탕 드래그: 영역 다중 선택
Shift + 클릭: 개별 추가 선택
모서리 핸들 드래그: 노드 크기 조절
노드 드래그 중 Alt: 정렬 스냅 임시 해제
🛠️ 도구 기능 (상단 버튼)
⚠ 검증 F8고립 노드·막힌 경로·무한루프 자동 탐지. 문제 노드를 빨간 글로우로 강조 📋 코드리뷰 F9플로우를 의사코드(pseudocode)로 자동 변환. 루프·분기·고립 노드 시각화 ▶ 로직 트레이스 F10시작 노드 선택 후 변수값 입력 → 단계별 실행. 조건식 자동 평가, 무한루프 감지 ⌨ 단축키 F1127개 액션의 키 바인딩 커스터마이즈. 프리셋(기본/Figma/Vim) 및 JSON 내보내기 지원 📁 가져오기JSON 파일 또는 텍스트 붙여넣기로 플로우 불러오기 💾 내보내기현재 플로우를 JSON으로 내보내기 (복사 또는 파일 저장) 📸 캡처캔버스를 PNG 또는 SVG로 내보내기 🕑 내역최대 50단계 Undo/Redo 이력 목록 확인 및 이동
▶ 로직 트레이스 사용법
1. F10 또는 헤더 ▶ 버튼 클릭
2. 시작 노드를 선택 (시작 타입 노드가 기본값)
3. 감지된 변수 패널에 테스트할 값 입력 (⚠ 표시는 미입력 변수)
4. ▶ 시작 버튼 클릭 → 단계별 실행
5. 분기점에서 조건이 자동 평가됨. 평가 불가 시 선택지 버튼이 나타남
6. 자동 모드 체크 시 조건 없는 분기는 첫 번째 경로로 자동 진행
※ 엣지 레이블이 비어있으면 레이블 없는 경로가 else(기본) 경로로 처리됩니다
